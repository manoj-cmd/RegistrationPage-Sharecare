# Sharecare Registration Form

## Introduction
This project is a replication of the Sharecare registration form using Node.js, React, Material-UI, and optionally Next.js.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd sharecare-registration/client

2. **Install dependencies:**

   ```sh
   npm install
   ```

3. **Start the React application:**

   ```sh
   npm start
   ```

### Usage

The frontend application will be available at `http://localhost:3000`.

### Testing

The project uses Jest and React Testing Library for frontend testing.

1. **Run tests:**

   ```sh
   npm test
   ```
